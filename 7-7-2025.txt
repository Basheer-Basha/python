11.Can I access a key using a value in a dictionary.
A.   No, you can't directly access a key using its value in a dictionary. However, you can
   iterate over the dictionary items to find the key:

   for key, value in dict.items():
    if value == desired_value:
        print(key)

12.Can I have duplicate values and keys in a dictionary? What happens if I wanted try to duplicate
   key in a dictionary?
A.  Values:
             Yes, dictionaries can have duplicate values. Multiple keys can have the same value.
       Ex: dict={"a":1,
                 "b":2,
                 "c":1}
           print(dict)   
          output:{'a': 1, 'b': 2, 'c': 1}

    Keys:
             No, dictionaries cannot have duplicate keys. If you try to create a dictionary with 
          duplicate keys, the last key-value pair will override any previous ones.   

       Ex: dict={"a":1,
                 "a";2,
                 "b":3}  
           print(dict)
          output: {'a': 1,'b': 3}
